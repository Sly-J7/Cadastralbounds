Introducing Cadastral Bounds

This is a web tool designed to facilitate the conversion of geographical coordinates into a two-dimensional XY map format, making it easier for users to visualize and manipulate spatial data. The platform supports users in generating shape files for specific concession areas, which can be utilized in various mapping and geographic information system (GIS) applications.

Key Features:

  • Coordinate Conversion:

    Users can input latitude and longitude values, which the website processes to generate corresponding XY coordinates for which users can also download the XY data format conversion in excel format. This feature is essential for translating geographic data into a flat, Cartesian coordinate system.

  • Shape Creation:

    The platform allows users to define specific shapes (e.g., polygons, lines) based on the XY coordinates. Users can draw their concession area boundaries through provision of geographical coordinates, which can represent land , resource extraction zones, or other relevant geographic features.

  • Shape File Generation:

    Once the shape is created, the website provides an option to export the shape as a shapefile (.shp). This format is widely used in GIS software and allows for easy integration into mapping projects.

  • Data Validation:

    To ensure accuracy, the site includes validation checks for input coordinates helping users avoid common errors during data entry.

  • Applications:

    Ideal for drawing and evaluating concession areas for environmental studies, urban planning, resource management (creating area boundaries for exploration and exploitation of minerals) and any field requiring precise geographic delineation of areas.

  • Add Row:
    Use this feature to add more rows to the table as your data exceeds the provided limit.



  • Double Click feature:
    Double click on any input area to share same data across columns in the table. This improves a quick way to manipulate data for processing.

    • Use the first pop-up box to insert the number( it can be empty to remove data from table cells ) you want to share across columns

    • The second second pop-up box is used to insert the first row number from which you want to share data from.

    • Use the last pop-up box to insert the last row number you want to share the data to



  • Download Excel:
    Use this download excel feature to download the excel table in .xls extension. The excel data includes the geographic coordinates and the x-y data format that can be uploaded to ARCMAP. Moreover, this makes it easy to store, share, and analyze your geographic data in a familiar spreadsheet environment.



  • Generate Chart:
    Visualize your geographic data with this feature. After entering your coordinates, you can create custom shapes that represent specific areas, such as concession zones or project boundaries. This feature allows for enhanced spatial analysis and better decision-making by providing a clear visual representation of your mapped areas.



  • Download Shapefile:
    Once you have created your desired shape for the concession area, use this feature to download a shapefile compatible with GIS software. This enables integration into your existing mapping projects and allows for more spatial analysis and manipulation of your data.



  • Pillar Identification:
    This feature allows users to define reference points or pillars within the mapped area, providing essential context for navigation and spatial analysis. By establishing these reference markers, you can easily identify key locations within your concession area.
